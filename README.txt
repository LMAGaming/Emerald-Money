# Dependencies
- **[CIT Resewn Continuation](https://modrinth.com/mod/cit-resewn-continuation)**

# About
Retextures most of the emerald items and blocks to look like money or coins.

## Features
- Texture of the emerald changes depending on the amount held [This is done by using the CIT Resewn Continuation mod].
- All emerald items and blocks are renamed to fit with the texture packs theme.

## Download
**[Download on Modrinth](https://modrinth.com/resourcepack/emerald-money)**
